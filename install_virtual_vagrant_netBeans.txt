VBox是开源的，而且功能强大，可以在 Linux/Mac 和 Windows 主机中运行，并支持在其中安装 Windows (NT 4.0、2000、XP、Server 2003、Vista、Win7)、DOS/Windows 3.x、Linux (2.4 和 2.6)、OpenBSD 等系列的操作系统。
硬件和操作系统和软件要求：

1)CPU:最低主频266MB，建议P3 1GHz以上，2核以上； 内存:最小128MB，建议512MB； 硬盘:最小空闲空间600MB，建议空闲空间5GB； 操作系统:必须是Windows NT内核操作系统，建议Windows 2000 SP2以上版本。
2)win8和10 pc-bsd：CPU必须支持VT-x或AMD-V 虚拟化技术功能 suse9/10.0禁止vt-x。其他相关资料链接https://www.virtualbox.org/wiki/Guest_OSes；
3）如果要用64位虚拟系统或者是Hyper-V虚拟机，那么CPU要支持虚拟化； 所以现在的电脑基本上都可以安装虚拟机的；

第一步：下载virtualbox：这是我的5.1.30版本 https://pan.baidu.com/s/1d7tMMJ1hl9oEdDLbs0_5nw；你也可以在官网是下载最新版本 https://www.virtualbox.org/wiki/Downloads；
下载linux的虚拟镜像文件：（注意：镜像文件都很大,请保留足够的空间）我这提供三个版本：lubuntu18.04：https://pan.baidu.com/s/1fgJ4IneIhwYrdYxazHSn6w；ubuntu16.04：https://pan.baidu.com/s/1Kxt0ehZVisPbf9DCh81Nxw；ubuntu14.04：https://pan.baidu.com/s/1KV-6RgFMyezM0eD5Rd_2yg；
第二步：安装virtualbox（可以查看流程图片）：
1）新建一个装虚拟机的文件vbox；
2）将下载好的virtualbox以及linux镜像文件倒到vbox中（这是我按照本人习惯，你也可以不用，只要在用时能找到即可）；
3）安装virtualbox（没有什么难度，按图安装，按顺序下一步即可）；
4）但安装完virtualbox后，先点击virtualbox上的设置，选择储存按钮，在控制器IDE上添加我们下载好的linux镜像文件；点击ok；
5）点击启动按钮：安装ubuntu；之后可以默认下一步；当到最后时（屏幕中安装步骤的小点都显示），输入你的用户名及密码；点击继续；开始安装；这时时间会很长，请耐心等待。
6）重启电脑；完成；
7）验证安装：进入安装的虚拟机系统界面，进入终端；输入命令sudo dmidecode | grep Product（如果没安装的话，请按照sudo apt install dmidecode）,接下来就会显示Product name:virtualBox，即安装虚拟机成功。
第三步：设置共享文件夹（win和虚拟机linux）：
1）点击虚拟机界面设置->共享文件夹->添加分享文件夹（右上角添加文件图标）；
2）在虚拟机界面上点击设备->安装增强功能（点击安装）；
3）进入到安装增强功能文件夹在终端运行VBoxLinuxAddtion.run；命令：./VBoxLinuxAddtion.run;
4）在终端中运行命令：sudo mount -t vboxsf 共享文件夹名称（在设置页面设置的） 挂载的目录；
5）点击你共享的文件夹，即可看到win上面共享的文件了；
6）当你关闭虚拟机后，虚拟机共享的文件是关闭的，你需要重新在终端中输入sudo mount -t vboxsf 共享文件夹名称（在设置页面设置的） 挂载的目录；当然，终端所在的目录文件夹是需要统一的，因此最好在安装增强功能文件中；
此外：关于虚拟机的网络链接：虚拟机默认的网络是NAT格式，及将虚拟机作为一个软件通过主机链接到外部网络，因此，NAT模式没有ip，他的ip是主机的ip；而我们可以将网络设置为桥接模式和仅主机可见模式；桥接模式是将虚拟机作为一个独立计算机，拥有独立的ip，通过主机连接外部网络；仅主机可见模式是仅虚拟机与主机间的网络连接，不具备链接外部网络，但可以通过安装代理软件等共享网络来进行联网；
可参考：https://www.cnblogs.com/xuan52rock/p/5295069.html；
在虚拟机上验证是否连接网络，可通过命令ping www.baidu.com(其可谓任何一个网址)来进行验证；如果没有网络，会提示网络解析错误；如果成功，则网络为会持续显示数据；

对于 Linux 平台，安装文件的扩展名为 .sh。对于这些平台，您需要使用以下命令来执行安装文件：chmod +x <installer-file-name>。键入 ./<installer-file-name> 以运行安装程序。

接受默认的 NetBeans IDE 安装目录，或者指定其他目录。
注：安装目录必须为空，并且在运行安装程序时使用的用户配置文件必须具有此目录的读/写权限。

（仅适用于全部或 Java EE 包。）接受要与 NetBeans IDE 一起使用的默认 JDK 安装，或者从下拉列表中选择其他安装。如果安装向导找不到要与 NetBeans IDE 一起使用的兼容 JDK 安装，则表明未在默认位置安装 JDK。在这种情况下，指定安装的 JDK 路径，然后单击“下一步”，或者取消当前安装。在安装所需的 JDK 版本后，您可以重新启动安装。
注：如果 JDK 版本早于建议的 JDK 8，请从 Java SE 下载页下载并安装最新的 JDK 更新，然后重新启动 NetBeans IDE 安装程序。

单击“下一步”。

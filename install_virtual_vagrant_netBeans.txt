vagrant install：
vagrant download url：https://www.vagrantup.com/downloads.html
linux :sudo apt install vagrant -y
vagrant运行的命令：
准备：准备好一个vagrantfile文件
vagrant up（此命令为启动vagrantfile文件生成虚拟机）
vagrant ssh （此命令为进入虚拟机）
以下为通过iso镜像制作个人box；
下载centos的minimal iso镜像文件，资源https://www.centos.org/ 中minimal iso选项；
运行vbox虚拟机进行安装；
注意：（将安装生成的文件与iso放在一个文件夹下如centosfile）
cd centosfile
git init
vagrant package —base centos75
vagrant box add centos75 package.box
以上为生成box
git init
vagrant init centos75 （此命令是根据box生成vagrantfile）
vagrant up
注意：当运行vagrant up命令结束后，在虚拟机vbox上点击show（显示），来显示并操作安装的系统；
以上为运行box

netBeans install：
netBeans  download  url：https://netbeans.org/
jdk download url：https://www.oracle.com/technetwork/java/javase/downloads/java-archive-downloads-javase7-521261.html
安装资料url：https://netbeans.org/community/releases/82/install_zh_CN.html
Microsoft Windows 和 Linux
安装软件：

下载完成后，运行安装程序。
对于 Windows，安装程序可执行文件的扩展名为 .exe。双击安装文件便可运行它。
对于 Linux 平台，安装文件的扩展名为 .sh。对于这些平台，您需要使用以下命令来执行安装文件：chmod +x <installer-file-name>。键入 ./<installer-file-name> 以运行安装程序。
如果下载了全部或 Java EE 包，则可以对安装进行定制。请执行安装向导的欢迎页上的以下步骤：
单击“定制”。
在“定制安装”对话框中进行选择。
单击“确定”。
在安装向导的欢迎页中，单击“下一步”。
在“许可证协议”页中，查看许可证协议，单击接受复选框，然后单击“下一步”。
在 NetBeans IDE 安装页中，执行以下操作：
接受默认的 NetBeans IDE 安装目录，或者指定其他目录。
注：安装目录必须为空，并且在运行安装程序时使用的用户配置文件必须具有此目录的读/写权限。

（仅适用于全部或 Java EE 包。）接受要与 NetBeans IDE 一起使用的默认 JDK 安装，或者从下拉列表中选择其他安装。如果安装向导找不到要与 NetBeans IDE 一起使用的兼容 JDK 安装，则表明未在默认位置安装 JDK。在这种情况下，指定安装的 JDK 路径，然后单击“下一步”，或者取消当前安装。在安装所需的 JDK 版本后，您可以重新启动安装。
注：如果 JDK 版本早于建议的 JDK 8，请从 Java SE 下载页下载并安装最新的 JDK 更新，然后重新启动 NetBeans IDE 安装程序。

单击“下一步”。
如果打开了 GlassFish Server Open Source Edition 4.1.1 安装页，请接受默认安装目录，或者指定其他安装位置，然后单击“下一步”。
如果您要在 Apache Tomcat 安装页上安装 Apache Tomcat，请接受默认安装目录，或者指定其他安装位置。单击“下一步”。
在“概要”页上，执行以下操作：
确保要安装的组件列表正确。
如果要检查更新中心中是否存在可能的更新，请选中“检查更新”复选框。
确保系统上有足够的空间可用于安装。
单击“安装”开始进行安装。
在“安装完成”页中，请根据需要提供匿名使用数据，然后单击“完成”。
注：如果您遇到问题而无法成功完成软件的安装，请参见疑难解答，该部分针对影响安装过程的未解决问题进行了描述并提供了建议的解决方法。

注：要使用已安装在计算机上的 GlassFish Server Open Source Edition 和/或 Apache Tomcat，请在 NetBeans IDE 中，单击工具菜单中的服务器。此时将会显示添加服务器实例向导，该向导可帮助在 NetBeans IDE 中注册所需 Web 服务器的实例。
